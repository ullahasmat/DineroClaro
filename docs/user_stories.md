# User Stories

- As a gig worker, I want to know how much to set aside for taxes so I avoid surprises.
- As a new saver, I want an emergency fund target and a weekly action list I can check off.
- As a borrower, I want to know which debt to pay first and how much to allocate monthly.
- As a bilingual user, I want to switch between English and Spanish without losing context.
- As a demo judge, I want to see a health check endpoint and seeded data working end-to-end.
