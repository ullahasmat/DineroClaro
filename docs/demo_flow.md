# Demo Flow (5 steps)

1) Landing: explain "ask anything about your money" + CTA to start chat.
2) Onboarding: collect goal (save, debt, invest) and monthly budget; write to Supabase.
3) Chat: show ChatBox, provide 2 suggested prompts; AI returns steps and a simple budget split.
4) Dashboard: checklist + recommendation cards; mark items complete (Supabase upsert).
5) Follow-up: share link to language toggle and invite for feedback.
